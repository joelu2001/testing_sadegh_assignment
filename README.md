# testing_sadegh_assignment

Detta projekt är en del av kurserna i Geodetisk mätteknik (VGMF15, VGMA05, EXTA50) vid LTH och Helsingborgs Campus. Detta repository lagrar HTML-koden för GitHub Pages-webbplatsen som finns på denna URL: https://joelu2001.github.io/testing_sadegh_assignment/. Webbplatsen består av ett Traversprotokoll i vänster panel och ett stängt polygontåg i höger panel. Användaren har möjlighet att fylla i observerade mätningar i specifika rutor, beräkna resultaten vilket fyller de återstående tomma rutorna och skapar en graf av det stängda polygontåget.
